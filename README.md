# MatraCare AI  
Real-Time Maternal Risk Prediction and Decision Support System

---

## Overview
MatraCare AI is an intelligent decision-support system designed to identify high-risk pregnancies early and provide actionable recommendations. It helps improve maternal healthcare by enabling timely intervention, especially in low-resource environments.

---

## Problem Statement
Maternal mortality remains a major global challenge due to delayed identification of high-risk pregnancies and lack of real-time clinical decision support. Existing systems primarily focus on data collection rather than guiding timely actions.

---

## Solution
MatraCare AI analyzes key maternal health inputs such as blood pressure, blood sugar, age, and symptoms to:

- Predict risk level (Low / Medium / High)  
- Generate a risk score (0–100)  
- Explain contributing risk factors  
- Recommend next steps for timely intervention  

---

## Key Features
- Real-time risk prediction  
- Explainable AI (risk breakdown)  
- Risk score visualization  
- What-if scenario simulation  
- Smart action timeline  
- Emergency alert system  

---

## System Workflow
User Input → Risk Analysis → Risk Score & Level → Explanation → Recommended Action

---

## Tech Stack
- Frontend: React  
- Backend: Python (Flask / FastAPI)  
- Logic: Rule-based AI (extendable to ML)  
- Database: Firebase / PostgreSQL
